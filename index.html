Δ Amsterunner

Fun facts and trivia about running

The name Amsterunner is a contraction of the city 'Amsterdam' in The Netherlands, where I currently live, and the word 'runner'.




"We run, not because we think it is doing us good, but because we enjoy it and cannot help ourselves. The more restricted our society and work become, the more necessary it will be to find some outlet for this craving for freedom. No one can say, 'You must not run faster than this, or jump higher than that.' The human spirit is indomitable."

- Sir Roger Bannister, first man to break the four minute mile.

Why this website?

This site is built within the framework of Holberton School application process. I have chosen the topic of running since practicing this activity has helped me in various ways, beyond the initial fitness benefits.

Thank you Holberton School for giving me this opportunity to develop valuable skills by learning in the same environment as professionals, for giving me access to my own server and for allowing me to interact with fellow learners. I have enjoyed every step of building my website from scratch, using the command line, which makes me feel empowered like Elliot.
